# Remote Setup

After creating an empty public GitHub repository, run:

```bash
git remote add origin git@github.com:wangxiao8600/oathai-uncertain-future-archive.git
git push -u origin main
```

If you prefer HTTPS:

```bash
git remote add origin https://github.com/wangxiao8600/oathai-uncertain-future-archive.git
git push -u origin main
```

If the repository name differs, replace the final path segment accordingly.
